## Skills手法のトレンドと課題（2026年2月）

### 手法の根幹とトレンド
2026年2月のSkills（スキル）研究は、従来のTool Use（ツール使用）を包含し、それを超えた上位概念として急速に確立されつつある。大きなトレンドは3点に集約される。第一に、スキルを単なるAPI呼び出しではなく、推論・判断・実行を含む「再利用可能な行動パターン」として体系化する動きの加速。SoK論文やサーベイ論文が相次いで発表され、スキルのアーキテクチャ、獲得方法、セキュリティまでを統一的に整理する試みが本格化した。第二に、強化学習を活用したスキルの自律的な獲得・進化（Skill Evolution）の実現。エージェントが経験を通じてスキルを再帰的に洗練・拡張し、タスクの複雑さに応じてスキルライブラリを自己成長させるアプローチが提案されている。第三に、スキルの汎用性と移植性を客観的に評価するベンチマークの登場であり、スキルが多様なタスクやドメインを横断してどの程度機能するかを定量的に測定する基盤が整い始めた。

### 課題傾向
主な課題は、スキルの構成可能性（Composability）と安全性の両立である。個々のスキルが単体で安全であっても、複数スキルの組み合わせにより予期しない振る舞いが発生しうる点が指摘されている。また、スキルの獲得プロセスにおいて、エージェントがどのような経験からどのようなスキルを抽出すべきかの指針が未確立であり、スキルの粒度（細かすぎると冗長、粗すぎると汎用性を失う）の最適化が技術的課題として残っている。さらに、スキルの共有・流通におけるセキュリティリスク（悪意のあるスキルの混入、プロンプトインジェクション等）への対策や、スキルの性能を公平に比較するための標準的な評価フレームワークの確立が急務となっている。

### 論文紹介

[SoK: Agentic Skills — Beyond Tool Use in LLM Agents](https://arxiv.org/abs/2602.20867)

エージェントの「スキル」概念を体系的に整理したSystematization of Knowledge論文。従来のTool Useを超えて、スキルを推論・計画・実行を含む高次の行動単位として定義し、その獲得・管理・評価に関する既存研究を横断的に分析。スキル研究の全体像を把握するための基盤的な文献。

[Agent Skills for Large Language Models: Architecture, Acquisition, Security, and the Path Forward](https://arxiv.org/abs/2602.12430)

LLMエージェントにおけるスキルのアーキテクチャ、獲得手法、セキュリティを包括的にサーベイした論文。スキルの設計パターンを分類し、学習ベース・経験ベースの獲得方法を比較するとともに、スキル共有時のセキュリティリスクと対策を論じている。

[SKILLRL: Evolving Agents via Recursive Skill-Augmented Reinforcement Learning](https://arxiv.org/abs/2602.08234)

強化学習を用いてエージェントのスキルを再帰的に進化させるフレームワーク。エージェントが獲得したスキルを基本ブロックとして新たなスキルを構築し、タスクの複雑さに応じてスキルライブラリを階層的に拡張する。スキルの自律的な成長と再利用性の実現を目指す。

[Agent Skills: A Data-Driven Analysis of Claude Skills for Extending Large Language Model Functionality](https://arxiv.org/abs/2602.08004)

Claude Skillsを対象としたデータ駆動型の分析。実際のユーザー利用データからスキルの使用パターン、成功率、失敗モードを定量的に解析し、どのようなスキル設計がLLMの機能拡張に効果的かを実証的に明らかにしている。

[SkillsBench: Benchmarking How Well Agent Skills Work Across Diverse Tasks](https://arxiv.org/abs/2602.12670)

エージェントスキルの汎用性を多様なタスクにわたって評価するベンチマーク。特定ドメインで獲得されたスキルが異なるドメインのタスクにどの程度転移するか、スキルの組み合わせによる性能変化などを体系的に測定し、スキルの実用性を客観的に評価する枠組みを提供。
